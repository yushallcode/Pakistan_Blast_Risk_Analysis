Heuristic Risk Algorithm
Density Score
max 40
+5 per incident in the cell, capped at 40. Rewards zones with concentrated attack history.
Depth Score
max 20
+3 per Quadtree subdivision level, capped at 20. Deeper cells = tighter clustering = higher localised risk.
Recency Score
max 25
+4 for incidents within 2 years, +2 for older. Recent activity weighted double.
Severity Score
max 15
+0.1 per person killed in the cell, capped at 15. Human cost factored into overall risk.
Risk Score = min( Density + Depth + Recency + Severity, 100 )
About the Project

What makes this different

Most risk dashboards rely on machine learning models whose decision-making is opaque. This system uses a mathematically transparent heuristic: every risk score is derivable from four documented components. The Quadtree recursively partitions Pakistan's geography based on incident density — areas with more events get subdivided further, giving finer resolution exactly where it matters.

Technical Stack

  • Custom Quadtree spatial index in pure Python
  • pandas for data cleaning & aggregation
  • matplotlib for static map generation
  • tkinter GUI with interactive query modes
  • Trend analysis: 3-window (3yr) escalation detection
  • Demo mode with province-centre coordinates
  • Risk profile reports per geographic cell
